Managing SSH Keys
-----------------

You can manage one or more SSH keys that can be added to an appliance template.

To add an SSH key: 

	1. Click on the ``My Account`` tab and go to SSH Keys.
	2. Click ``Add``.
	3. Enter the key name and the key details.
	4. Click ``Create``.
