For Windows
-----------

Run the following command::

	c:\Python27> .\Scripts\easy_install.exe uforge-python-sdk


Upgrading the SDK
~~~~~~~~~~~~~~~~~

To upgrade an already installed SDK, run::

	c:\Python27> .\Scripts\easy_install.exe --upgrade uforge-python-sdk
