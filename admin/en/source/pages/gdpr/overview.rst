GDPR Compliance
----------------

General Data Protection Regulation (GDPR) is a regulation that requires businesses to protect the personal data and privacy of EU citizens. In the context of GDPR, USS provides some basic information to help UForge administrators list user data held by UForge as well as delete the data associated to a specific user as needed.
